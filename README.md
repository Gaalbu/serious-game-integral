# Slay the integral

[cite_start]Um jogo educacional *roguelike* e *dungeon crawler* de turnos focado no ensino prático de cálculo integral[cite: 2, 18]. [cite_start]Desenvolvido para estudantes que desejam aprender matemática de forma interativa e punitiva na medida certa[cite: 5].

## 🎯 Objetivo
[cite_start]Derrote inimigos e chegue ao Chefe Final para "Integrá-lo" completamente[cite: 13]. Sobreviva utilizando seus conhecimentos matemáticos!

## ⚔️ Como Jogar
* [cite_start]**Combate:** Cada inimigo possui uma "Função de Resistência"[cite: 9]. [cite_start]Equipe armas e resolva as integrais propostas na tela para atacar[cite: 9, 10].
* [cite_start]**Risco e Recompensa:** Acertar o cálculo causa dano direto[cite: 11]. [cite_start]Errar a integral faz sua arma falhar ou causa dano a você mesmo[cite: 11, 13].
* [cite_start]**Progressão:** Inimigos avançados exigem métodos mais complexos (como substituição ou integração por partes) para serem derrotados[cite: 10, 12].

## 🛠️ Tecnologias
* [cite_start]Python (Pygame) ou Unity[cite: 4].
